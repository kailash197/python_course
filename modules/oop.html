Module 5: Object-Oriented Programming (OOP)

In this module, you'll learn about classes, objects, attributes, methods, and inheritance.

class Robot:
    def __init__(self, name, color):
        self.name = name
        self.color = color
        self.battery = 100

    def move_forward(self):
        if self.battery > 0:
            print(f"{self.name} is moving forward.")
            self.battery -= 10
        else:
            print(f"{self.name} has no battery left!")
