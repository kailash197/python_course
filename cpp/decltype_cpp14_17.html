decltype in C++14/17

Definition

decltype is a keyword in C++ used to query the type of an expression at compile time. It enables type deduction without evaluating the expression.

Syntax

decltype(expression)

Returns the type of the expression without evaluating it.

Use Cases

  • Type deduction in templates
  • Declaring variables with complex types
  • Writing generic code
  • Robotics: deducing types from sensor APIs or control functions

Examples

int speed = 10;
decltype(speed) velocity = 20; // velocity is of type int

auto distance = 100.5;
decltype(distance) range = 200.0; // range is of type double

template<typename T1, typename T2>
auto add(T1 a, T2 b) -> decltype(a + b) {
    return a + b;
}

Best Practices

  • Use decltype to simplify complex type declarations.
  • Combine with auto for powerful type inference.
