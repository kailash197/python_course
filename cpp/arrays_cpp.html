Arrays in C++14/17

Arrays are a fundamental data structure in C++ used to store collections of elements of the same type. They are particularly useful in robotics for storing sensor data, control parameters, and more.

1. Definition and Syntax

An array is a fixed-size sequence of elements of the same type.

int sensorValues[5];

2. Types of Arrays

One-Dimensional Arrays

float distances[3] = {1.2, 2.5, 3.8};

Two-Dimensional Arrays

int grid[2][3] = {{1, 2, 3}, {4, 5, 6}};

Dynamic Arrays

Using pointers and dynamic memory allocation:

int* data = new int[10];

3. Usage in Robotics

  • Storing sensor readings (e.g., LIDAR, ultrasonic)
  • Maintaining control signal history
  • Representing grid maps or occupancy grids

4. Best Practices

  • Use std::array or std::vector for safer and more flexible arrays
  • Always check bounds when accessing array elements
  • Free dynamically allocated memory using delete[]

5. Code Example


#include <iostream>
#include <array>

int main() {
    std::array<float, 5> sensorData = {0.5, 1.2, 0.8, 1.5, 2.0};
    for (float value : sensorData) {
        std::cout << "Sensor reading: " << value << std::endl;
    }
    return 0;
}

This example uses std::array for safer array handling in C++14/17.
