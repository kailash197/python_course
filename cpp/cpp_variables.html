Variables in C++14/17

Definition

A variable in C++ is a named storage that programs can manipulate. Each variable in C++ has a specific type, which determines the size and layout of the variable's memory.

Types of Variables

  • int - Integer values
  • float, double - Floating-point numbers
  • char - Character type
  • bool - Boolean values (true/false)
  • auto - Type inference (C++11 and later)
  • const - Constant values

Syntax Examples

int age = 25;
float pi = 3.14;
char grade = 'A';
bool isPassed = true;
auto name = "Kailash";

Scope and Lifetime

Scope refers to the region of the program where the variable is accessible.
Lifetime refers to how long the variable exists in memory.

  • Local variables - Declared inside functions or blocks
  • Global variables - Declared outside all functions
  • Static variables - Retain value between function calls

Best Practices

  • Use meaningful variable names
  • Initialize variables before use
  • Prefer auto for type inference when it improves readability
  • Use const for values that should not change

Illustrative Code Snippets

// Example 1: Local variable
void greet() {
    std::string message = "Hello, World!";
    std::cout << message << std::endl;
}

// Example 2: Static variable
void counter() {
    static int count = 0;
    count++;
    std::cout << "Count: " << count << std::endl;
}

// Example 3: Const variable
const double gravity = 9.81;
