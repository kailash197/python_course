bool Data Type in C++14/17

The bool data type in C++ represents Boolean values: true and false. It is used primarily in conditional expressions and logical operations.

Definition

bool is a built-in data type that can hold one of two values:

  • true (typically represented as 1)
  • false (typically represented as 0)

Usage in Conditions

bool isReady = true;
if (isReady) {
    std::cout << "System is ready!" << std::endl;
}

Logical Operations

bool a = true;
bool b = false;

bool result = a && b; // false
result = a || b;       // true
result = !a;           // false

Conversions

Any non-zero value is implicitly converted to true, and zero is converted to false.

int x = 5;
bool flag = x; // true

x = 0;
flag = x; // false

Best Practices

  • Use bool for flags and conditions instead of int.
  • Prefer explicit comparisons for clarity: if (x == true).
  • Avoid using bool in arithmetic expressions.

Example

#include <iostream>

int main() {
    bool isOnline = true;
    if (isOnline) {
        std::cout << "Device is online." << std::endl;
    } else {
        std::cout << "Device is offline." << std::endl;
    }
    return 0;
}
