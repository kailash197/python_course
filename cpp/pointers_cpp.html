Pointers in C++14/17

1. What is a Pointer?

A pointer is a variable that stores the memory address of another variable.

int a = 10;
int* ptr = &a; // ptr holds the address of a

2. Pointer Syntax

  • * is used to declare a pointer.
  • & is used to get the address of a variable.

3. Pointer Arithmetic

Pointers can be incremented or decremented to traverse arrays.

int arr[] = {1, 2, 3};
int* p = arr;
p++; // points to arr[1]

4. Null Pointers

Use nullptr (introduced in C++11) to initialize a pointer that doesn't point to anything.

int* p = nullptr;

5. Smart Pointers (C++14/17)

Smart pointers manage memory automatically and help prevent memory leaks.

  • std::unique_ptr - sole ownership
  • std::shared_ptr - shared ownership
#include <memory>

std::unique_ptr<int> uptr = std::make_unique<int>(10);
std::shared_ptr<int> sptr = std::make_shared<int>(20);

6. Pointers in Robotics

  • Accessing sensor data from memory-mapped registers
  • Dynamic memory allocation for sensor arrays
  • Efficient data passing in real-time control loops
void readSensor(int* data) {
    // process sensor data
}

7. Best Practices

  • Always initialize pointers.
  • Use nullptr instead of NULL.
  • Prefer smart pointers over raw pointers.
  • Avoid dangling pointers and memory leaks.
