Floating-Point Types in C++14/17

Overview

Floating-point types are used to represent real numbers (numbers with fractional parts). In C++, the primary floating-point types are:

  • float - Single precision (typically 4 bytes)
  • double - Double precision (typically 8 bytes)
  • long double - Extended precision (typically 8, 12, or 16 bytes depending on the platform)

Type Modifiers

  • signed - Default for floating-point types; allows positive and negative values
  • const - Declares a variable whose value cannot be changed
  • volatile - Tells the compiler that the variable may be changed externally (e.g., by hardware)

Precision and Usage

Precision is critical in robotics for tasks like sensor data processing, PID control, and kinematics. Use float for memory-constrained systems and double for higher precision.

Examples in Robotics

// Sensor reading (e.g., temperature)
const float temperature = 36.7f;

// PID control values
double kp = 0.1, ki = 0.01, kd = 0.05;

// Volatile variable updated by an interrupt
volatile float imu_angle;

// Long double for high-precision calculations
long double trajectory_time = 12.34567890123456789L;

Best Practices

  • Use float for embedded systems with limited resources
  • Use double for general-purpose robotics applications
  • Use const for values that should not change (e.g., constants)
  • Use volatile for variables shared with hardware or interrupt routines

Conclusion

Understanding floating-point types and their modifiers is essential for writing efficient and accurate robotics software in C++14/17.
