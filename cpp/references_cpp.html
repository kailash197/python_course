References in C++14/17

Definition

A reference in C++ is an alias for another variable. Once a reference is initialized to a variable, it cannot be changed to refer to another variable.

Syntax

int a = 10;
int& ref = a; // ref is a reference to a

Differences from Pointers

  • References must be initialized when declared; pointers can be null.
  • References cannot be reseated; pointers can point to different objects.
  • References use the same syntax as the original variable; pointers use * and &.

Usage in Robotics

References are commonly used in robotics for:

  • Parameter Passing: Efficiently pass large sensor data structures to functions without copying.
  • Actuator Control: Modify motor control parameters directly via reference.
  • Real-time Systems: Avoid unnecessary memory allocations by using references.

Code Example

void updateMotorSpeed(int& speed) {
    speed += 10;
}

int main() {
    int motorSpeed = 50;
    updateMotorSpeed(motorSpeed);
    // motorSpeed is now 60
}

Best Practices

  • Use const references when you don't want to modify the original variable.
  • Avoid dangling references by ensuring the referenced object outlives the reference.
  • Use references for performance-critical code where copying is expensive.
